\chapter*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

Для решения задачи генерации текста на данный момент наилучшим вариантом является архитектура трансформеров, однако данная модель может быть избыточной для тривиальных задач, потому что использует высокие вычислительные мощности и требует большого объема обучающих данных.

В ходе выполнения данной научно-исследовательской работы были решены следующие задачи:
\begin{itemize}
    \item[---] Обозначены основные вехи развития методов генерации текста;
    \item[---] Формализована задача генерации текста;
    \item[---] Перечислены методы решения задачи генерации текста;
    \item[---] Произведено сравнение рассмотренных методов. 
\end{itemize}

В рамках работы были выполнены все поставленные задачи. Цель работы была достигнута.
