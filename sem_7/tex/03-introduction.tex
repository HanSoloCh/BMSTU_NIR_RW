\chapter*{ВВЕДЕНИЕ}
\addcontentsline{toc}{chapter}{ВВЕДЕНИЕ}


Кластеризация данных является одной из ключевых задач анализа данных и машинного обучения, направленной на выявление скрытой структуры в наборах объектов без использования априорно заданных классов. Методы кластеризации широко применяются в задачах интеллектуального анализа данных, распознавания образов, обработки естественного языка, анализа изображений, биоинформатики и других областях, где требуется группировка объектов по степени их сходства.

Целью данной научно-исследовательской работы является анализ существующих методов кластеризации данных. Для достижения поставленной цели необходимо решить следующие задачи:
\begin{itemize}
    \item[---] провести анализ предметной области кластеризации данных;
    \item[---] рассмотреть существующие меры расстояний;
    \item[---] рассмотреть основные методы кластеризации;
    \item[---] выделить критерии сравнения методов кластеризации;
    \item[---] сравнить рассмотренные методы по выделенным критериям;
    \item[---] рассмотреть основные методы оценки качества кластеризации.
\end{itemize}
